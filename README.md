# Plano de Qualidade da Netflix

![Capa do Projeto](https://upload.wikimedia.org/wikipedia/commons/thumb/6/69/Netflix_logo.svg/1024px-Netflix_logo.svg.png)

# Sobre o Projeto

<h4>Grupo formado com o objetivo de criar um Plano de Qualidade da plataforma de streaming NETFLIX para aprimorar a experiência do usuário. Iremos trabalhar com a metodologia FURPS e tambem com gerencia de risco A Netflix é a líder mundial do entreterimento digital, e para manter essa posição, é essencial fornecer uma experiência expicional aos seus clientes. </h4>
